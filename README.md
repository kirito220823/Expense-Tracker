# Expense Tracker

This is a simple expense tracker built with HTML, CSS, and JavaScript to help you manage your expenses.

## Features
- Add and track expenses and income
- View your balance at a glance
- Simple and easy-to-use interface

## How to Use
1. Clone the repository to your local machine.
2. Open `index.html` in your browser.
3. Start adding your transactions!
